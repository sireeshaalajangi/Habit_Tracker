# 🔥 Habit Tracker Web App

A productivity-based Habit Tracker built using HTML, CSS, and JavaScript.  
This app helps users track daily activities, monitor time spent, and improve consistency.

---

## 🚀 Features

- ✅ Add preset and custom activities (Gym, Study, Coding, etc.)
- ⏱️ Track time spent on each activity
- ✅ Mark tasks as completed
- ✏️ Edit activities
- ❌ Delete tasks
- 🔄 Reset daily data
- 📊 Show completed vs pending tasks
- 🔥 Display total time spent per day
- 💾 Data stored using localStorage (no data loss on refresh)

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript (Vanilla JS)
- LocalStorage
